# 🎓 Smart Student Management System

A professional Python-based Student Management System developed using SQLite, Pandas, Matplotlib, and ReportLab. This project helps manage student records efficiently with a secure admin login, analytics, reports, graphs, and database backup.

---

## 🚀 Features

- 🔐 Secure Admin Login
- ➕ Add Student
- 👀 View Students
- ✏️ Update Student
- ❌ Delete Student
- 🔍 Search Student
  - By ID
  - By Name
  - By Grade
  - By Minimum Marks
  - By Maximum Marks
- 📊 Student Dashboard
- 📈 Graph Analytics
  - Bar Chart
  - Line Chart
  - Pie Chart
- 📄 Generate PDF Report
- 📁 Export Data to CSV
- 🏆 Student Ranking
- 💾 Database Backup
- ♻️ Database Restore
- 📝 Activity Logging
- ✅ Input Validation

# ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/smart-student-management.git
```

### Open Project

```bash
cd smart_student_management
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python main.py
```
